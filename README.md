# 웹프로그래밍 수업 요약

## 1. 수업 개요
- 웹의 기본 구조와 동작 원리 학습
- 클라이언트-서버 모델 이해
- 기초적인 웹 애플리케이션 제작

## 2. 주요 학습 내용
### (1) HTML
- 웹 페이지의 뼈대 구성
- 주요 태그: `<html>`, `<head>`, `<body>`, `<a>`, `<img>` 등

### (2) CSS
- 웹 페이지의 스타일링
- 선택자, 박스 모델, 레이아웃, 반응형 디자인

### (3) JavaScript
- 동적인 동작 구현
- DOM 조작, 이벤트 처리, 함수 활용

### (4) 서버 프로그래밍 기초
- HTTP 프로토콜 이해
- 요청과 응답 구조 학습

## 3. 학습 목표
- 웹 페이지 제작의 기본기 습득
- 디자인과 기능을 통합한 웹 구현
- 기본적인 웹 애플리케이션 개발 가능
